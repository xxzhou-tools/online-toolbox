# 在线工具箱 - 多合一实用工具集合

一个简单、实用、免费的多合一工具网站，包含多种常用在线工具。

## 包含的工具

1. **密码生成器** - 生成安全、随机的强密码
2. **JSON 格式化** - 格式化、压缩 JSON 数据
3. **Base64 编解码** - Base64 编码和解码转换
4. **字数统计** - 统计文字、字符、段落数量
5. **颜色转换器** - HEX、RGB 颜色格式互转
6. **时间戳转换** - Unix 时间戳与日期时间互转

## 本地运行

```bash
cd tools
python3 -m http.server 8000
```

然后访问 http://localhost:8000
